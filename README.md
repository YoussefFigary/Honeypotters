# 🌍 World Tourism Explorer

A web application that allows tourists to discover and explore amazing travel destinations from around the world — with detailed descriptions for every sightseeing location.

---

## ✨ Features

- 🗺️ Browse tourism destinations from around the world
- 🏛️ View detailed descriptions for each sightseeing location
- 🔍 Search and explore attractions by location
- 📱 Clean and user-friendly interface

---

## 🛠️ Tech Stack

| Technology | Role |
|---|---|
| **JavaScript** | Frontend & backend logic |
| **Node.js** | Server-side runtime |
| **MongoDB** | Database for storing locations & descriptions |
